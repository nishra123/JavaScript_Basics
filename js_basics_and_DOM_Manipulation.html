Kay zhalyavar kay zhala pahije, hyala event mhantat, for example:

1. use of onclick="function()"

- on clicking that button, it will call function : function clicked(){ console.log("The button was clicked!") }

2. use of window.onload=function()

it calls the function() when the page/window is loaded

3. use of .addEventListener('element', function())

it adds event to the element, for example: firstcontainer.addEventListener('click', function(){ document.querySelectorAll('.container')[1].innerHTML = " Button dabaya Re! " console.log("Clicked on container") }) // it will get executed when you click let prevHTML = document.querySelectorAll('.container')[1].innerHTML; firstcontainer.addEventListener('mouseover', function(){ document.querySelectorAll('.container')[1].innerHTML = prevHTML; console.log("Mouse on container") }) // it will get executed when you bring mouse on it firstcontainer.addEventListener('mouseout', function(){ document.querySelectorAll('.container')[1].innerHTML = " Button dabaya Re! " console.log("Mouse out of container") }) // it will get executed when you bring mouse out of it firstcontainer.addEventListener('mouseup', function(){ console.log("Mouse up when clicked on container") }) firstcontainer.addEventListener('mousedown', function(){ console.log("Mouse down when clicked on container") })
